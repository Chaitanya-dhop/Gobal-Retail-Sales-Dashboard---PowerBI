# Global Retail Sales Dashboard (Power BI)

## Overview
This project showcases an interactive Global Retail Sales Dashboard built using Power BI. It provides insights into sales performance, profit, and regional trends to support data-driven decision-making.

## Key Features
 KPI Metrics: Total Sales, Profit, Profit Margin, Return Rate
 Regional Sales Analysis
 Top 10 Products by Sales
 Monthly Sales Trend Analysis
 Segment & Category Breakdown
 Sales vs Target Comparison
 Interactive Filters (Region & Year)
## Tools & Technologies
- Power BI Desktop
- Power Query (Data Cleaning & Transformation)
- DAX (Data Analysis Expressions)
## Insights
- The central region has the highest sales contribution
- Technology category performs best
- Consumer segment dominates sales
- Sales show seasonal trends and fluctuations

Global_Retail_Sales_Dashboard.pbit

 To build an interactive dashboard for analyzing business performance and generating actionable insights.
